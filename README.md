# 「盗月」窃月瓷坊

景徳鎮の伝統工芸 × 東方Project のオリジナル陶磁器グッズ制作サークルの公式ウェブサイト

## 🏠 ページ構成

- **ホーム (home.html)** - サークル紹介、購入リンク、最新情報
- **工芸紹介 (syoukai.html)** - 代表的な工芸と作品
- **ニュース速報 (news.html)** - イベント情報と新商品ニュース
- **オーダーメイド相談 (odermade.html)** - カスタム制作相談フォーム
- **私たちについて (about.html)** - サークル詳細情報

## 🎨 技術

- **フロントエンド**: HTML5, CSS3
- **レスポンシブ対応**: モバイル・タブレット対応済み
- **フォーム**: Formspree連携

## 📝 ローカル開発

任意のHTMLサーバーで実行可能:

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

ブラウザで `http://localhost:8000` を開く
